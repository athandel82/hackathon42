## Overview

The model is a small AUTOSAR **Electronic Door Control (EDC)** demo. Two identical
`Door` components (left and right) publish lock/open status and accept lock
commands. A central `DoorControl` reads both doors' status, aggregates them into a
`CombinedStatus`, commands the door locks, and drives an LED via the `IoHwAb`
hardware-abstraction service. The `EDC` composition wires these together and
exposes `CombinedStatus`, which the `EDCEcuExtract` SYSTEM maps to four network
system signals в†’ I-Signals в†’ I-PDUs. Platform data types come from the
`ArcCore` blueprint package.
